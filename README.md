# پیش‌بینی قیمت طلا با ترکیب LSTM و XGBoost و FinBERT

## 📌 معرفی پروژه
این پروژه یک سیستم ترکیبی برای پیش‌بینی قیمت طلا است که از داده‌های تاریخی، شاخص‌های اقتصادی و تحلیل احساسات اخبار استفاده می‌کند.

## 🛠️ تکنولوژی‌ها
- **Python 3.8+**
- **XGBoost** برای پیش‌بینی بازده روزانه
- **LSTM** برای پیش‌بینی قیمت سری‌زمانی
- **FinBERT** برای تحلیل احساسات اخبار
- **LangChain** برای تولید گزارش متنی
